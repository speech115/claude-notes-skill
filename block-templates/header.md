# Header Template

Main context writes this directly (small, deterministic). Save to `$WORK_DIR/header.md`.

Use the **content-based title** from step 2 (NOT the raw title from notes-runner, which is often just the file name or first transcript line).

## Format

```markdown
# [Content-based title] (полный конспект)

> Формат: [type], [duration estimate from prepare_report], [participants]
> Спикеры: [names from speakers.txt with brief context]
> Тема: [topic from first 50 lines]
```
