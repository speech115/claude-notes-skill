# Header Template

Main context writes this directly (small, deterministic). Save to `$WORK_DIR/header.md`.

Use the **content-based title** from step 2 (NOT the raw title from notes-runner, which is often just the file name or first transcript line).

## Format

```markdown
# [Content-based title]

> [2-3 sentence content summary in Russian]

**Автор:** [real person if known]
**Формат:** [лекция / интервью / разбор / воркшоп / групповая сессия]
**Тема:** [one-line core topic]
**Длительность:** [duration estimate]
**Источник:** [YouTube link or source label]

## Главная рамка автора
[Short paragraph explaining the author's core lens, what they push against, and what practical frame they offer.]
```
